# Как выложить сайт на GitHub Pages

1. Создай репозиторий на GitHub.
2. Загрузи файл `index.html` в корень репозитория.
3. Открой Settings -> Pages.
4. В Source выбери `Deploy from a branch`.
5. Ветка: `main`, папка: `/root`.
6. Нажми Save.
7. Через 1-3 минуты появится ссылка на сайт.

Если хочешь, можешь отправить клиенту только ссылку.
На телефоне сайт можно открыть и выбрать «Добавить на экран».
